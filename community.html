Về Trang Web

Trang Web Tự Cải Thiện

Trang web Pharos Vietnam được thiết kế để dễ dàng cập nhật nội dung mà không cần kỹ năng lập trình phức tạp. Toàn bộ dữ liệu — tin tức, dự án hệ sinh thái, hướng dẫn — đều nằm trong một file duy nhất.

Bạn có thể đóng góp nội dung mới bằng cách liên hệ với admin hoặc trực tiếp cập nhật data file nếu bạn là thành viên quản trị.

🔄 Cách cập nhật nội dung:
  1. Mở file js/data.js
  2. Thêm tin tức mới vào mảng news
  3. Thêm dự án mới vào mảng ecosystem
  4. Cập nhật meta.lastUpdated
  5. Website tự động cập nhật toàn bộ trang!
